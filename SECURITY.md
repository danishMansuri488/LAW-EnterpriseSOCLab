# Security Policy

## Supported Versions

The latest version of this repository is actively maintained.

| Version | Supported |
|----------|-----------|
| 1.0.x | ✅ |

---

## Reporting a Security Issue

If you identify a security issue, documentation error, or technical inaccuracy within this repository, please open a GitHub Issue describing the problem.

Please include:

- A clear description of the issue
- Steps to reproduce (if applicable)
- Suggested improvement (optional)

---

## Scope

This repository is an educational SOC laboratory designed for cybersecurity learning, detection engineering, and threat hunting.

No production systems, sensitive data, or live enterprise environments are used.

---

## Responsible Disclosure

Please do not publish issues that could negatively impact other users before allowing reasonable time for review and correction.

---

## Contact

For repository-related questions or security concerns, please use the GitHub Issues section.
