# LAW-EnterpriseSOCLab

Enterprise Security Operations Center (SOC) laboratory designed to simulate enterprise-scale security monitoring, detection engineering, threat hunting, and incident investigation using Microsoft Sentinel, Splunk Enterprise, Active Directory, Sysmon, and Windows Security telemetry.

---

## Overview

LAW-EnterpriseSOCLab is a comprehensive enterprise security monitoring laboratory that replicates key Security Operations Center (SOC) workflows within a controlled environment. The project demonstrates how security telemetry can be collected, analyzed, correlated, and investigated to detect suspicious activity across Windows infrastructure.

The environment combines Microsoft Sentinel, Splunk Enterprise, Windows Server Active Directory, Sysmon, and Windows Event Logs to emulate enterprise authentication, endpoint, and identity monitoring. Detection logic, alert validation, threat hunting, and investigation workflows are implemented to demonstrate practical SOC operations aligned with modern blue-team methodologies.

Rather than focusing solely on SIEM deployment, the project emphasizes detection engineering principles, structured investigation techniques, and repeatable analytical workflows that support security monitoring and incident response.

---

## Objectives

- Design an enterprise SOC monitoring laboratory
- Collect and analyze Windows Security Events
- Centralize security telemetry within SIEM platforms
- Develop detection logic for common attack techniques
- Validate detections using simulated attack scenarios
- Perform proactive threat hunting across collected telemetry
- Document investigation workflows and analytical findings
- Map detections to the MITRE ATT&CK Framework

---

## Project Scope

The laboratory simulates a Windows enterprise environment where authentication events, endpoint activity, Active Directory operations, and security logs are continuously monitored.

Core focus areas include:

- Detection Engineering
- Security Monitoring
- Threat Hunting
- Incident Investigation
- Windows Event Analysis
- Active Directory Monitoring
- Authentication Monitoring
- Alert Validation
- SIEM Operations
- MITRE ATT&CK Mapping
