# Contributing

Thank you for your interest in contributing to LAW-EnterpriseSOCLab.

## Contribution Guidelines

Contributions that improve the quality, accuracy, or educational value of this repository are welcome.

Examples include:

- Documentation improvements
- Microsoft Sentinel detection enhancements
- KQL query improvements
- Analytics Rule improvements
- Lab architecture updates
- Bug fixes
- Typographical corrections

---

## Pull Requests

When submitting a pull request, please ensure:

- Documentation is clear and technically accurate.
- Markdown formatting is consistent.
- KQL queries are validated before submission.
- Detection logic is tested where applicable.
- New content aligns with the project's scope.

---

## Issues

If you encounter a bug or have a suggestion, please open a GitHub Issue including:

- A clear description of the issue
- Expected behavior
- Steps to reproduce (if applicable)
- Suggested improvement (optional)

---

## Code of Conduct

Please maintain a professional, respectful, and collaborative environment when contributing to this project.
